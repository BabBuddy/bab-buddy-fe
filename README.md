# Bab-Buddy-Fe

### Requirements

- Node.js 18+
- Yarn

### Quick Start

```bash
# Clone and install
git clone https://github.com/BabBuddy/bab-buddy-fe.git
cd bab-buddy-fe
yarn

# Start development server
yarn dev
```

### Commands

```bash
yarn dev    # Start dev server
yarn build  # Build for production
yarn start  # Run production server
yarn lint   # Check code quality
```
